## Project Instructions

- Build this project as a Transmission torrent manager for macOS.
- Optimize the UI for mobile first, but keep it usable on desktop layouts.
- Use shadcn/ui component patterns for interface work.
- The upstream Transmission codebase is available locally in `transmission/`.
- Transmission already includes a web interface in `transmission/web/`; use that code as a reference for RPC usage, flows, and UI behavior before inventing new patterns.
- Treat `transmission/` as reference material unless a task explicitly asks you to modify the upstream code.
